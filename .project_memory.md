## Project Overview
- **Name**: Stock_Price_Calculator
- **Purpose**: A desktop application (Electron + HTML/CSS/JS) to calculate stock target prices based on intrinsic values.
- **Tech Stack**: Electron, Node.js, Vanilla HTML/CSS/JS.

## Naming & Formatting Conventions
- **Project Naming**: All project names must be capitalized for each word, and connected with underscores (`_`). Example: `Stock_Price_Calculator`.
- **Variables & Code**: Generally follow standard JS conventions unless specified otherwise, but project/folder names strictly follow the `Word_Word_Word` underscore format.

## Design & Style Preferences
- **UI Theme**: Light color scheme (`--bg-light: #f4f4f5`, `--card-bg: #ffffff`). Compact, modern, premium UI.
- **Layout**: Two-column (左右版面), calculation form on the left, history on the right. 
- **Window Management**: 
  - Fixed size (Width: 520, Height: 680) to bypass macOS memory and enforce a compact, "hugging" layout.
  - Floating above other apps (`alwaysOnTop: true`).
- **Icon Style**: Minimalist Flat Apple Style (like modern iOS flat design). 
  - *Icon Sizing (CRITICAL)*: Ensure exactly **15% transparent padding** around the core logo so it aligns perfectly with native macOS dock icons.
  - *Icon Processing*: When removing backgrounds, use an edge-inward flood fill to avoid hollowing out inner shapes of the same color.
